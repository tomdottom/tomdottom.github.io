Learning Python the language

  1. Codecademy Python

Codecademy provides many excelent introductory guides including this one on Python. It will walk you through Python's fundemental concepts, and syntax of the programming language.

All coding is done through Codecademy's interactive online interface. You don't have to install anything and can do the course on almost an PC with a browser and internet connection.

  1. Google's Python Class

You may find youself repeating some stuff from Codecademy Python, but Google's class will take you a little bit deeper into some of Python's builting packages.

It also comes with a two lectures to which will help cement your understanding of the basics.

Lastly there are four excercises for you to work through.

Installing Python

  1. The Hitchhiker’s Guide to Installing Python
The Hitchhiker’s Guide to Python contains lots of guides for all levels of Python. But amoungst its most useful is a guide on how to install Python for the most popular operating systems. If your OS isn't listed, you probably understand enough not to need this guide.

Essential Python Tools

  1. virtualenv

This is really one of the most essential tools when working with Python. It stops a lot of pain in the long run by preventing conflicts between your induvidual projects and even system installed packages.

Confused? Then go read The Hitchhiker’s Guide to Virtual Environments which will guide you through using virutal environments and try to explain to you what they are along the way.

If this doesn't satisfy you questions then head over to the virtualenv documentation
